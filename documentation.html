Flow Estimator - A Uniform, Steady Open Channel Flow Calculator for QGIS

Description

This plugin calculates flow and velocity using the Manning equation for uniform, steady flow. If you've never heard of the Manning equation or taken a course in open channel hydraulics you won't understand what this tool is doing.

If you are familar with open channel hydraulics and want a quick refresher check out this information from the University of Delaware:

http://web.archive.org/web/20160305025907/udel.edu/~inamdar/EGTE215/Open_channel.pdf

A great reference for Mannings n values can be found from the USGS here:

http://pubs.usgs.gov/wsp/2339/report.pdf

User Defined Cross Section

The user defined cross section option allows loading a tab or space delimited text file without a header. The first column should contain station values and the second column should contain elevation values.

Usage Considerations

Make sure the steady, uniform flow assumption is valid.

When sampling a DEM cross-section be sure to make the cross-section perpendicular to flow at a location that is representative of the stream or drainage channel.

When sampling DEM cross-sections and slopes, make sure the DEM is in the same CRS as the project CRS (but you knew that already right!).
